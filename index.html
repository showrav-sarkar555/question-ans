Question 1 (16th Batch TT)

a) What is a polynomial Equation and properties of it with example?

A polynomial equation is an expression consisting of variables and coefficients, involving only the operations of addition, subtraction, multiplication, and non-negative integer exponentiation of variables. The general form is:

$$P(x) = a_n x^n + a_{n-1} x^{n-1} + \ldots + a_1 x + a_0 = 0$$ where $a_n \neq 0$.

Properties:

  • Degree: The highest power of the variable (n) determines the degree.
  • Roots: A polynomial of degree n has exactly n roots (real or complex), counting multiplicities (Fundamental Theorem of Algebra).
  • Continuity: Polynomials are continuous and smooth (differentiable) everywhere.
  • End Behavior: The sign of the leading coefficient ($a_n$) determines the behavior of the graph as $x$ approaches $\pm\infty$.
  • Multiplicity: If a root is repeated, it is called a multiple root, and the graph touches the x-axis at that point.

Example: $x^2 - 5x + 6 = 0$ is a polynomial equation of degree 2. Its roots are $x = 2$ and $x = 3$.

Question 2 (16th Batch TT)

a) When does the Newton Raphson method fails and how to solve the issues?

Failure Cases:

  • Zero Derivative: If $f'(x_n) = 0$ at any iteration, the method fails as the iteration formula becomes undefined.
  • Poor Initial Guess: If the initial guess $x_0$ is not sufficiently close to the root, the method may diverge or oscillate.
  • Multiple Roots: The method converges very slowly for multiple roots (where $f(x) = 0$ and $f'(x) = 0$).
  • No Root: If the function has no real root, the method will not converge.
  • Oscillation: It can enter an infinite cycle oscillating between two values.

Solutions:

  • For zero derivative: Use a different initial guess or switch to a bracketing method like bisection or false position for a few iterations.
  • For poor initial guess: Use a graphical method or a bracketing method to find a good initial approximation.
  • For multiple roots: Use the modified Newton-Raphson method: $$x_{n+1} = x_n - m \frac{f(x_n)}{f'(x_n)}$$ where $m$ is the multiplicity of the root.
  • For oscillation: Use a method that guarantees convergence, like the bisection method, or implement a damping factor.

Question 3 (14th Batch Final)

a) Write down the algorithm for the false position iteration method.

False Position (Regula Falsi) Algorithm:

  1. Start: Choose initial guesses $a$ and $b$ such that $f(a) \cdot f(b) < 0$.
  2. Iterate: For n = 0, 1, 2, ... until convergence:
    • Calculate $c = \frac{a \cdot f(b) - b \cdot f(a)}{f(b) - f(a)}$
    • Calculate $f(c)$
    • If $|f(c)| < \epsilon$ (a very small tolerance), then stop. $c$ is the root.
    • Else, check the sign:
      • If $f(a) \cdot f(c) < 0$, then the root lies in $[a, c]$. Set $b = c$.
      • Else, the root lies in $[c, b]$. Set $a = c$.
  3. End

Question 6 (12th Batch Final)

a) Define Newton Raphson method with algorithm.

The Newton-Raphson method is an iterative method for finding successively better approximations to the roots (or zeroes) of a real-valued function.

Newton-Raphson Algorithm:

  1. Choose an initial guess $x_0$ (close to the true root).
  2. For n = 0, 1, 2, ... until convergence:
    • Calculate $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$
    • If $|x_{n+1} - x_n| < \epsilon$ (a specified tolerance), then $x_{n+1}$ is the root. Stop.

a) Write down the drawbacks of bisection method.

  • Slow Convergence: Compared to other methods like Newton-Raphson, it converges slowly (error reduces by roughly half each iteration).
  • Requires Bracketing: Needs an initial interval [a, b] where the function changes sign.
  • Cannot find roots of even multiplicity: If a root is tangent to the x-axis (e.g., a double root), the function may not change sign, and the method fails.
  • Inefficient: Requires many function evaluations to achieve high accuracy.

b) What is the bisection method and what is it based on?

The bisection method is a bracketing root-finding method that repeatedly bisects an interval and then selects a subinterval in which a root must lie for further processing. It is based on the Intermediate Value Theorem, which states that if a continuous function $f$ has values of opposite sign inside an interval [a, b], then it has at least one root in that interval.

Question 9 (11th Batch Final)

a) What do you mean by order of convergence? / What is a Transcendental equation with an example?

Order of Convergence (p):

A measure of how fast a sequence $x_n$ converges to its limit $x^*$. It is the largest positive number p satisfying:

$$\lim_{n \to \infty} \frac{|x_{n+1} - x^*|}{|x_n - x^*|^p} = \mu$$

where $\mu$ is the rate of convergence. A higher order means faster convergence.

Example: Newton-Raphson has quadratic convergence (p=2).

Transcendental Equation:

An equation containing a transcendental function (e.g., trigonometric, exponential, logarithmic) of the variable. These equations often have an infinite number of roots or no algebraic solution.

Example: $e^x + \sin x = 0$

c) Write down the algorithm for solving the system of equations using Gauss-Jordan elimination.

Gauss-Jordan Elimination Algorithm:

  1. Form the augmented matrix [A | B].
  2. For k = 1 to n (n is the number of equations):
    • Pivot: Find the row with the largest absolute value in column k (from row k to n) and swap it with row k.
    • Normalize: Make the pivot element 1: $R_k = R_k / a_{kk}$
    • Eliminate: For all other rows i (i ≠ k), make the element in column k zero: $R_i = R_i - a_{ik} * R_k$
  3. After processing all columns, the augmented matrix becomes [I | X], where X is the solution vector.

d) Write down the main difference between Gauss-Jacobi method and Gauss-Seidel method.

The main difference is in how they use the updated values:

  • Gauss-Jacobi: Uses the values from the previous iteration only to compute all new values for the next iteration. It does not use any updated information during the current iteration.
  • Gauss-Seidel: Immediately uses the newly computed values in the current iteration to update the next variables. It uses the most recent information available, which generally leads to faster convergence.
